CampusCoPilot - Implemented Features Summary
=============================================

Here is the crux of the core features and subsystems implemented in the platform:

1. Core Platform & Premium UI/UX 
---------------------------------
• 3D Interactive Design: Fully integrated `Background3D` component utilizing React-Three-Fiber for subtle, modern 3D backgrounds.
• Premium Light Theme: A cohesive "Cream White" aesthetic using advanced glassmorphism components with dark slate text for high readability and premium feel.
• Smooth Animations: Leverages Framer Motion and GSAP for seamless page transitions, modal pop-ins, and scroll animations. 
• Floating Animated Navigation: A sleek, modern floating action button (FAB) in the bottom right corner containing the app's navigation paths.
• Fully Responsive Layouts: Adapts effectively from desktop to mobile screens. 

2. Intelligent Study Planner & Roadmaps (The "Strategy Brain")
----------------------------------------------------------------
• AI Document Parsing: Ability to upload Syllabus/Curriculum PDFs; the backend automatically extracts structural topics and subtopics.
• Subject Personalization: Users can label subjects with Difficulty (Strong/Okay/Weak) to deeply influence backend schedule generation.
• Smart Time Assignment (LLM-based): The AI generates realistic time-to-completion, adjusting duration based on user weaknesses and AI-judged complexity. 
• Realism Constraints: Generates roadmaps with a strict 6–8 hours daily limit. 
• Mandatory Breaks Injection: The schedule intelligently inserts natural resting blocks (Coffee/Tea/Lunch) to prevent burnout.
• Custom Events Integration: Students can append both recurring events (like classes) and deadline-driven single events (like assignments) to the timeline. 

3. Study Squads (Social & Collaborative Learning)
--------------------------------------------------
• Squad Creation & Joining: Students can create topic-focused study groups, generating a 5-digit Invite Code for friends to join securely. 
• Classmate Discovery: A built-in search engine allows searching for peers on the platform by University and Branch/Major. 
• Squad Dashboards: Live tracker of members and shared subjects.

4. Micro-Tutor Mode (Targeted Learning)
---------------------------------------
• Dynamic Quizzes: Auto-creates graded tests consisting of MCQs, True/False, and short-answer questions tailored to any topic. 
• AI Flashcards: Generates flippable 3D flashcards summarizing key points from user query points.
• Automated Resource Retrieval: Directly aggregates YouTube tutorial links, Khan Academy pages, Wikipedia articles, and Google Scholar papers for any requested subject. 
• Assessment History: Saves past scores and tracks historical understanding.

5. Focus Execution & Anti-Distraction (The "Execution Engine")
---------------------------------------------------------------
• Smart Pomodoro Timer: A highly visible, non-blocking timer built into the application. 
• AI "Focus Plans": When starting a session, the LLM provides an explicit breakdown of what to accomplish in that exact 25/50 minute block (e.g. mini-goals).
• Statistics & Analytics: Dashboard graphs showing study consistency, upcoming tests, and Pomodoros completed. 

6. Additional Capabilities
--------------------------
• Multi-modal Notes Scanner: Interfaces to upload handwritten notes, integrating text extraction.
• Gamification & Portfolio: Achievement tracking showing unlocked badges and earned 'learning credits'.
• Comprehensive Profile Data: Tracks target grades, desired career roles, and core learning constraints.
